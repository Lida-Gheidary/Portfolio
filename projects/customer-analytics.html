Customer Analytics Dashboard

Self-service analytics tool for exploring insurance marketing campaign performance

Python Plotly Dash Pandas Data Visualization
Live Demo View Code

An interactive dashboard enabling marketing teams to explore campaign performance data independently. Built with Python and Plotly Dash, it provides real-time filtering and dynamic visualizations for data-driven decision making.

Objective

  • Build a self-service analytics tool for marketing teams to explore campaign performance
  • Enable stakeholders to filter and analyze data without analyst intervention
  • Demonstrate ability to create custom dashboards beyond traditional BI tools

Dashboard Features

Interactive Filters

  • Sales channel selection
  • Policy type filtering
  • Coverage level options
  • Real-time updates

Key Metrics

  • Total customer lifetime value
  • Customer count
  • Monthly premium revenue
  • Average CLV per customer

Visualizations

  • CLV breakdown treemap
  • Campaign response pie chart
  • Dynamic insights panel
  • Export to CSV functionality

Dataset

  • Source: IBM Watson insurance customer dataset for marketing/CRM analytics
  • Size: 9,134 customer records with demographics, policy information, and campaign responses
  • Key Fields: Customer segments, CLV, monthly premiums, sales channels, response (Yes/No indicating campaign acceptance)

Technical Approach

  • Built Python-based dashboard using Plotly Dash for real-time interactivity
  • Implemented callback functions for dynamic filter updates across all visualizations
  • Designed responsive layout ensuring usability on desktop and mobile devices
  • Deployed to Render for public access without requiring local environment setup

Why This Project Matters

  • Demonstrates ability to go beyond drag-and-drop BI tools (Looker, Tableau) for custom solutions
  • Shows Python programming skills applied to real-world business analytics
  • Proves understanding of stakeholder needs: self-service, clarity, and flexibility
  • Portfolio piece showcasing technical depth for data analyst roles

Tools & Technologies

Python, Plotly Dash, Pandas, NumPy | Deployed on Render
